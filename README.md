Financial tracking app for Android with utilization of device's geoinformation

Features include:
- Adding and viewing accounts, transactions
- Adding location info to transactions, current location name suggestions
- Cool pie charts for transactions by categories, transactions by places for specified date range
- Accounts balance graphs for specified date range
- Total accounts balance summary for specified date range
- Overview of transactions with geoinformation on map
- Synchronization of accounts and transactions with Monobank (Ukrainian bank)
